# OIBSIP_PythonProgramming_task02
BMI Calculator ⚖️

A simple BMI (Body Mass Index) calculator using GUI.

📂 Location: `/BMI_Calculator`

🛠️ **Technologies:**
- `tkinter`

🧮 **Functionality:**
- Takes height (in cm) and weight (in kg)
- Calculates BMI
- Displays health status (Underweight, Normal, Overweight, Obese)
